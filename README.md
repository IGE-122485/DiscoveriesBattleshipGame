# Battleship

IGE (Grupo TP06-9) | Francisco Brites Antunes: 123027 | Diogo Almeida da Silva: 123018 | António Sousa Koenders: 122485 | Francisco Loureiro Nunes de Carvalho: 122984

Tipos de Navios: Porta-Aviões, Navio de 4 canhões, Navio de 3 canhões, Navio de 2 canhões e Submarino

Regras do jogo: A Batalha Naval é um jogo de estratégia para dois jogadores onde o objetivo é afundar a frota adversária escondida num grelha de 10X10. Cada jogador posiciona os seus navios na horizontal/vertical sem se tocarem, alternando tiros por coordenadas. Ganha quem afundar todos os barcos primeiro. 

Tipos de embarcações existentes na era dos Descobrimentos: https://pt.wikipedia.org/wiki/Lista_de_navios_da_Marinha_Portuguesa
